Propósito del portafolio

Este portafolio tiene como propósito la recopilación estructurada de diferentes prácticas y proyectos desarrollados a lo largo de la materia de CNO V – Seguridad Informática.

A través de este espacio digital se documentan los conocimientos técnicos, el razonamiento aplicado y la evolución del aprendizaje, permitiendo evidenciar de forma clara el proceso académico y el desarrollo de competencias en el área de la seguridad informática.

Enfoque del curso

El curso está enfocado en áreas clave de la seguridad informática, incluyendo la identificación de vulnerabilidades, pruebas de penetración y el análisis de riesgos en sistemas informáticos.

El portafolio de evidencias permite al estudiante visualizar su propio progreso, reflexionar sobre las soluciones implementadas y consolidar el aprendizaje mediante la documentación técnica de cada práctica.

Perfil del estudiante

Soy estudiante de la Universidad Politécnica de San Luis Potosí, con un interés particular en la ciberseguridad y el análisis de datos.

A lo largo de mi formación académica he desarrollado diversos proyectos tecnológicos, que incluyen sistemas de control de inventarios, dashboards interactivos para visualización de información, aplicaciones web y soluciones orientadas al manejo y análisis de datos.

Mi interés se centra en comprender cómo la seguridad, los datos y el desarrollo de software convergen para crear sistemas eficientes, seguros y confiables.

Información técnica

Durante el desarrollo de este portafolio y los proyectos académicos se han utilizado diversas tecnologías y lenguajes de programación, entre ellos:

Python para análisis de datos y automatización, Power BI para dashboards y visualización, HTML, CSS y JavaScript para desarrollo web, así como C++ y otros lenguajes vistos a lo largo de la carrera.

Este portafolio se encuentra publicado en línea mediante GitHub Pages, utilizando Git como sistema de control de versiones, lo que permite llevar un seguimiento claro de los cambios y la evolución del proyecto.
